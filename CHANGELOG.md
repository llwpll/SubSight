# Changelog

All notable changes to SubSight will be documented in this file.

## 0.1.0 - 2026-07-09

- Added the SwiftUI macOS app for local subscription tracking.
- Added local JSON persistence shared by the app and CLI.
- Added `subsightctl` for list, get, add, update, pause, resume, delete, summary, breakdown, rates, templates, import, and export workflows.
- Added CSV and JSON import/export.
- Added category/payment breakdowns, upcoming renewals, privacy mode, payment limits, end dates, and menu bar access.
- Added release packaging scripts for the app and CLI.
